# ▌稍后阅读

「稍后阅读」是《人升》的一项隐藏功能，但非常实用。

其原理是《人升》支持接收其他应用「分享」的信息，从而创建稍后阅读事项。


**《人升》可接收其他应用分享的信息，快速建立「稍后阅读」事项，**并默认勾选「学识」属性。

**支持大部分提供分享功能的应用：**如微博、知乎、浏览器、Twitter 等。

## 举例：知乎

1. 打开知乎的回答或文章，点击右上角菜单，再点击「更多」。

![](_media/tasks/read_it_later_01.jpg ':size=30%')

2. 找到并点击《人升》图标。

![](_media/tasks/read_it_later_02.jpg ':size=20%')

3. 跳转至新建事项界面后，点击创建按钮。

![](_media/tasks/read_it_later_03.jpg ':size=30%')


4. 在事项详情页，点击链接即可跳转回知乎对应页面。


![](_media/tasks/read_it_later_04.jpg ':size=30%')

---

## 举例：夸克浏览器

1. 使用夸克浏览器打开任意页面，点击「菜单」，再点击「分享」按钮。

![](_media/tasks/read_it_later_05.jpg ':size=30%')

2. 向左滑动至最右侧，选择「更多」。

![](_media/tasks/read_it_later_06.jpg ':size=30%')

3. 找到并点击《人升》图标，即可创建「稍后阅读」事项。

![](_media/tasks/read_it_later_07.jpg ':size=30%')
