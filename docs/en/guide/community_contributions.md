## Community Contributions 🤝

We are incredibly grateful to our community for building tools that extend LifeUp's capabilities. Below are some featured projects created by our users:

| Project | Description | Author |
| :--- | :--- | :--- |
| [lifeup-mcp](https://github.com/derekprovance/lifeup-mcp) | A Model Context Protocol (MCP) server that enables LLMs (like Claude) to interact with LifeUp via the SDK. | [@derekprovance](https://github.com/derekprovance) |
| [lifeup_mod_coins](https://github.com/klienkross/lifeup_mod_coins) | A community plugin that connects Obsidian and LifeUp, supporting to-do sync (assisted info filling), reflection sync, and viewing LifeUp data dashboards in Obsidian. *Note: this repository is currently Chinese-only.* | [@klienkross](https://github.com/klienkross) |

> [!TIP]
> **Built something cool?** If you've developed a tool, script, or integration for LifeUp, please let us know via GitHub Issues or email!
